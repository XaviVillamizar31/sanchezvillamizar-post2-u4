**Estudiante:** Xavi Villamizar
**Programa:** Ingeniería de Sistemas

## Descripción del proyecto

Este proyecto consiste en el desarrollo de un **formulario de registro de usuario** con validación completa del lado del cliente utilizando **HTML5, CSS3 y JavaScript**. Se implementa la **Constraint Validation API del navegador** junto con validaciones manuales mediante JavaScript para verificar los datos ingresados por el usuario. El sistema muestra **mensajes de error específicos por campo**, retroalimentación visual inmediata y controla el envío del formulario.

## Instrucciones de ejecución

1. Clonar o descargar el repositorio.
2. Abrir la carpeta del proyecto en **Visual Studio Code**.
3. Hacer clic derecho en el archivo **index.html**.
4. Seleccionar **"Open with Live Server"**.
5. El navegador abrirá automáticamente el formulario de registro para probar las validaciones.
